def AxfcIRBuilder.AxfcIRBuilder.build_ir (   self,
  path 
)

This method is used to build AXI IR.

1) it builds a naive IR using the given input model. 2) it checks the IR nodes to be executed in hardware-manner. 3) it finds AIXH IR blocks. each block consist of several AIXH IR nodes. 4) it performs the liveness analysis for resolving the input and output of the blocks.
